## ðŸš¨ RED FLAGS TO AVOID

If your feature has ANY of these, it's likely non-compliant:

- [ ] Uses words like "best", "top", "leading", "expert"
- [ ] Shows star ratings or scores
- [ ] Displays client testimonials
- [ ] Ranks lawyers in any order except alphabetical
- [ ] Involves payment for visibility
- [ ] Uses celebrity/influencer promotion
- [ ] Shows success rates or case statistics
- [ ] Creates "featured" or "premium" profiles
- [ ] Implements paid referral system
- [ ] Uses promotional language
